# 智能体学习

## 一、学习目标

系统学习智能体相关知识，并服务个人科研、自动化任务和项目开发。

## 二、学习内容

1. MCP
2. RAG
3. Agent框架
4. 工具调用
5. 任务规划
6. 记忆管理
7. 自动化工作流
8. GitHub Actions
9. 自动生成日报、周报、论文整理和邮件任务

## 三、学习方式

主要通过B站系统课程学习。  
学习目标不是只看概念，而是能做出可运行的小型工作流。
