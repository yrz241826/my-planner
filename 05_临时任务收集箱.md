# 05_临时任务收集箱.md

## 文件作用

本文件用于暂存临时出现的新任务，例如：

- 导师临时安排
- 课程作业
- 比赛通知
- 项目临时需求
- 面试任务
- 论文修改
- 材料提交
- 突发事务

临时任务不一定都要进入长期规划。  
Hermes需要判断它属于临时任务、已有项目的新任务，还是全新长期方向。

---

## 临时任务处理规则

| 类型 | 处理方式 |
|---|---|
| 一两天内完成的小任务 | 写入当前任务池和今日记录 |
| 属于已有项目的新任务 | 写入当前任务池，并同步更新对应projects文件 |
| 全新项目任务 | 写入当前任务池，并新建projects文件 |
| 影响长期方向的任务 | 写入重要决策记录，必要时更新总规划 |
| 暂时不确定是否要做 | 保留在本文件，等待确认 |
